# Spam Email Classifier

This is a simple project to understand how text classification works using machine learning.

The goal is to classify messages as spam or not spam.

---

## What I Did

- Converted text into numerical form  
- Trained a basic classification model  
- Tested it on a sample input  

---

## Why This Project

I wanted to connect what I learned in NLP (classification, text processing) to something practical.

---

## Note

This is a basic implementation focused on understanding the process rather than optimization.
